1. Download the affilinet Extension
  2. Install the Extension using Chrome or Firefox Add-on Manager
  3. Open your browser and click on Extension icon to provide Publisher ID and Webservice Password as well as select the country platform your publisher was created on
    Your Publisher Webservice Password can be located within your Publisher Account under
    Solutions Web Services Access Data
    4. Please ensure that you use the Publisher Webservice Password and NOT the Product Data Webservice Password.
  4. You are ready to go!
